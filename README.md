# 🕷️ Web Scraping de Produtos com Python

Projeto desenvolvido para automatizar a coleta de dados de produtos de um site utilizando Selenium.

## 🚀 Funcionalidades

- Acessa automaticamente o site
- Coleta:
  - Nome dos produtos
  - Preços
- Salva os dados em arquivo CSV

## 🛠️ Tecnologias

- Python
- Selenium

## 📂 Estrutura

- `src/scraper.py` → código principal
- `data/precos.csv` → dados coletados
